# ILLAM THULIR FABRICS

Welcome to ILLAM THULIR FABRICS, your premier destination for premium fabrics for every occasion. Our collection features a wide range of traditional and contemporary textiles, perfect for all your needs.

## Project Overview

This project is a fashion website built using React, TypeScript, Firebase, and Cloudinary. It showcases a variety of textile products, including cotton sarees, silk sarees, dress materials, and more, all designed to celebrate the rich textile heritage of Tamil Nadu.

## Features

- **Elegant Design**: A premium and mobile-responsive design that enhances user experience.
- **Product Categories**: Explore our diverse range of products:
  - Cotton Sarees
  - Silk Sarees
  - Dress Materials
  - Chudithars
  - Kurtis
  - Men's Wear
  - Kids Wear
  - Blouse Materials
  - Dhotis
- **WhatsApp Enquiry**: Easily inquire about products through WhatsApp with pre-filled messages including product details.
- **Admin Dashboard**: Manage products and categories seamlessly with a dedicated admin interface.

## Getting Started

To get started with the project, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd illam-thulir-fabrics
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**:
   Create a `.env` file in the root directory and add your Firebase configuration.

4. **Run the Application**:
   ```bash
   npm start
   ```

5. **Access the Application**:
   Open your browser and navigate to `http://localhost:3000`.
